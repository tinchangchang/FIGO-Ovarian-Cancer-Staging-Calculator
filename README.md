# FIGO Ovarian Staging Calculator
Offline-capable FIGO 2014 staging calculator for epithelial ovarian, fallopian tube, and peritoneal cancer.
Reference: Renz M, Friedlander M, Berek JS. Cancer of the ovary, fallopian tube, and peritoneum: 2025 update. Int J Gynecol Obstet. 2025;171(Suppl. 1):6-35. DOI: 10.1002/ijgo.70282.
This calculator is a staging aid only. Confirm final FIGO stage with complete operative, imaging, pathological, cytological, and multidisciplinary information before clinical use.
